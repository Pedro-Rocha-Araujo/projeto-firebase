# TodoList Firebase 🔥
### O presente projeto foi desenvolvido em react, e integrado com o Firebase, tendo as seguintes funcionalidades:
* Cadastrar e fazer login
* Listar, ascionar, deletar e editar tarefas
* Tarefas exclusivas para cada login
* Responsividade e abordagem mobile first
